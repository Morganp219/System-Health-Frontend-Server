#System Health Frontend Server
This server is to aid the creation of a Frontend-based System Status Health Monitor.

This was created with Node.JS, Express, SystemInformation, Cors, Socket.io

To use this project
Pull straight from Docker Hub:
```
docker run -p 3001:3001 morganp219/system-monitor-challenge
```
OR 

1. Clone this repository to your machine
2. Move into the respoitories folder
3. Run in Terminal: ```docker compose up --build```
